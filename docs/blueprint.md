# **App Name**: Karaoke Queue Master

## Core Features:

- Song Request Submission: Patrons can submit song requests via a mobile-friendly form including song title, artist name, and optional special announcements.
- Tipping Interface: Patrons can tip the KJ/band using predefined amounts or a custom amount via a mobile interface.
- Real-time Queue View: Display the current song and upcoming queue in real-time for patrons on their mobile devices.
- Admin Queue Management: KJ/Admin can manage the song queue, mark songs as 'Now Playing' or 'Finished,' and remove songs via a protected admin interface.
- User role management: Secure admin page only accessible by admin role

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a nightlife atmosphere and musicality.
- Background color: Dark gray (#2B2B2B), building on the existing #242424 to ensure readability in dim environments; desaturated purple tint (20% saturation).
- Accent color: Bright orange (#FF9800) for interactive elements, tips, and notifications to draw attention and indicate action.
- Headline font: 'Belleza', sans-serif. Body font: 'Alegreya', serif.
- Patron View: Single, scrollable page with 'Now Playing' and upcoming queue. KJ/Admin View: Dashboard-style layout with controls for managing the queue.
- Use clear, minimalist icons to represent actions such as 'add song', 'tip', 'now playing', and 'finished'.
- Subtle transitions and animations to indicate changes in the queue and real-time updates, creating a smooth user experience.
